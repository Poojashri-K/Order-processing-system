# Order Processing System

## Description
This project manages and processes customer orders efficiently using priority-based order handling.

## Features
- Create and manage orders
- Priority-based order processing
- Order status tracking
- Simple and clean user interface

## Technologies Used
- HTML, CSS, JavaScript
- Node.js
- C++ (Data Structures concepts)

## How to Run
1. Clone the repository
2. Open the project in VS Code
3. Start the server and open the application
